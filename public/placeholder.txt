This directory is for static assets like images.

Place your images here:
- profile.jpg (your profile photo)
- project1.jpg (e-commerce project image)
- project2.jpg (task management app image)
- project3.jpg (weather dashboard image)
- project4.jpg (portfolio generator image)

You can use any image format (jpg, png, webp, etc.) and update the src attributes in the code accordingly.
